Abstract

This module of ZAMO (Zeri Art Market Ontology) models the most important events of the art market domain, stating the most relevant classes and properties.

attribute assignmentc back to ToC or Class ToC

IRI: https://w3id.org/zeri/ontology/zamo/events#AttributeAssignment

This class indicates a peculiar kind of "Event" in which an agent makes assertions on an "Object" instance. This assertion consists of expressing a specific attribute related to the object, such as the author or the previous owner (see "AttributeType"). As an example, art historians' expertises fall within this class.

has facilitatorop back to ToC or Object Property ToC

IRI: https://w3id.org/zeri/ontology/zamo/events#hasFacilitator

This property identifies any agent who takes an active part in an event of type "Transaction". A facilitator does not have a direct economic interest in the commercial transaction and should not be confused with the seller (see "hasSurrender") or the buyer (see "hasReceiver"). A typical case of facilitator in a transaction in this domain may be a scholar who suggests an agent to buy a specific object.
has super-properties
has participantop
has domain
transactionc
has range
